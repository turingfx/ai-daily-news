🚀 Q2 2026 重大功能更新 NEW
Anthropic 近期发布了多项重大功能，将 Claude Code 推向全新的 agentic engineering 阶段。
Dynamic Workflows - 动态工作流编排 FEATURE
Claude 现在可以跨数十到数百个后台 agent 编排工作，处理复杂任务。运行 /workflows 查看运行状态。使用 ! 前缀在 claude agents 中将 shell 命令作为后台会话运行。
Agent View (Research Preview) - Agent 仪表板 FEATURE
集中管理所有会话（运行中、阻塞或已完成）的仪表板。命令 claude agents 支持 JSON 输出、按目录筛选、重命名后台会话、固定会话保持活跃。
Claude Opus 4.8 支持 NEW
默认 High Effort (/effort xhigh)，Fast Mode 以 2 倍速率获得 2.5 倍速度。Auto Mode 现已在 Bedrock、Vertex 和 Foundry 上线（Opus 4.7/4.8）。设置 CLAUDE_CODE_ENABLE_AUTO_MODE=1 启用。
增强 Skills & Plugins FEATURE
.claude/skills 目录中的插件自动加载（无需 marketplace）。claude plugin init 快速创建插件，claude plugin prune 清理孤立依赖，claude plugin validate 验证清单。
Goal-Oriented Agents - 目标导向 Agent FEATURE
/goal [完成条件] 让 Claude 跨多轮工作直到目标达成，实时显示已用时间、轮数和 token 消耗。
🛠️ 改进与修复 Improvements
  • 全屏模式：改进渲染、鼠标支持、自动复制选中内容
  • Vim Mode：新增 visual mode (v) 和 visual-line mode (V)
  • Diff 渲染：大文件编辑性能提升，支持 j/k、PgUp/PgDn 键盘导航
  • MCP：远程 MCP 服务器自动重试瞬态故障，修复 OAuth 并发刷新问题
  • 安全：--dangerously-skip-permissions 绕过 .claude/、.git/ 等写入提示
  • WSL：修复图片粘贴、截图粘贴、拖拽图片等问题
  • CJK：修复 Windows Terminal 幽灵字符和宽字符布局问题
  • API：修复 Opus 4.8 thinking blocks 错误、不支持 effort 参数的模型 400 错误
