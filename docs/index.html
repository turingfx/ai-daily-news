Claude Code 更新日志

v2.1.91
🚀 Q1 2026 六大新功能
Anthropic 在 Q1 发布了六项重要功能：Remote Control（服务器/CI部署）、 Dispatch（任务感知路由）、Channels（发布/订阅消息）、 Computer Use（GUI自动化）、Auto Mode（自主执行）、 以及 AutoDream（规划循环）。
v2.1.91 最新版本 NEW
  • MCP 工具结果持久化覆盖 - 新增 _meta["anthropic/maxResultSizeChars"] 注解（最高500K），支持更大的结果如数据库schema
  • 多行深度链接支持 - claude-cli://open?q= 现在支持编码换行符（%0A）
  • 插件可执行文件 - 插件可在 bin/ 目录下发布可执行文件，并作为裸命令调用
  • Bug 修复 - 修复 --resume 时的转录链断裂、各终端的 cmd+delete、远程会话中的计划模式等问题
📅 2026-04-02 🔧 9 项变更
v2.1.90 上一版本 RECENT

  • 性能优化 - Bun 上更快的 stripAnsi，SSE 传输以线性时间处理大流帧
